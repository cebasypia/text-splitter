# Text Splitter

長文を Unicode コードポイント単位で指定した長さに分割し、各チャンクをワンクリックでコピーできる Web アプリケーションです。翻訳サイトなどの文字数制限があるサービスへの入力を効率化することを目的としています。

## 主な機能

- **正確な文字数カウント**: `String.length` ではなく、サロゲートペアに対応した Unicode コードポイント単位でカウント・分割します。
- **改行コードの正規化**: 入力テキストを内部で LF (`\n`) に正規化し、プラットフォームに依存しない正確な分割を実現します。
- **ワンクリックコピー**: 分割された各チャンクを即座にクリップボードにコピーできます。
- **自動更新**: テキストの入力や分割設定の変更に合わせて、リアルタイムで分割結果が更新されます。
- **プライバシー配慮**: 入力したテキストはサーバーに送信・保存されず、ブラウザ内のみで処理されます。

## 技術スタック

- **フレームワーク**: React 19
- **ビルドツール**: Vite 7
- **言語**: TypeScript 5
- **スタイリング**: Tailwind CSS 4
- **アイコン**: Lucide React
- **デプロイ**: GitHub Pages

## 開発環境の構築

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```

## ディレクトリ構成

- `src/components/`: UI コンポーネント（入力、設定、チャンク表示など）
- `src/hooks/`: カスタムフック（分割ロジック、クリップボード操作など）
- `src/utils/`: ユーティリティ関数（改行正規化、コードポイント分割など）
- `src/App.tsx`: メインエントリポイントと状態管理
